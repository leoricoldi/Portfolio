Projetos

Banco de Sangue - visual

O projeto Banco de Sangue é uma aplicação web desenvolvida para promover a doação de sangue no Brasil. O site apresenta uma interface simples, mas rica em conteúdo, destacando as tecnologias HTML5 e CSS3, oferecendo informações claras e acessíveis para incentivar novos doadores e facilitar o contato com os hemocentros.
